# Sales Dashboard (Power BI)

## About Project
This is a simple Sales Dashboard made using Power BI.  
It shows sales data in an easy way using charts and graphs.

---

## What I Did
- Created monthly sales trend (Jan to Dec)
- Showed category-wise sales (like Furniture)
- Compared online and offline sales
- Added basic KPIs for total sales

---

## Tools Used
- Power BI  
- Excel  
- DAX (basic)

---

## What I Learned
- How to clean data using Power Query  
- How to create charts and dashboards  
- How to use filters and slicers  

---

## How to Use
1. Open `.pbix` file in Power BI  
2. Click on filters to explore data
